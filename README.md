## 📖 About StockBet
**StockBet** brings a fresh, gamified twist to trading. Bet on real-time stock movements, monitor your portfolio, explore IPOs, and backtest quantitative strategies — all in one immersive platform.

---

## ✨ Features <a id="features"></a>
- **📊 Real-Time Market Data** — Powered by Yahoo Finance API  
- **💸 Bet on Stock Movements** — Go long or short, make the call!  
- **📱 Live Position Tracking** — Monitor your open bets with ease  
- **📂 Full Trading History** — Analyze your past performance  
- **🏦 Banking System** — Deposit & withdraw funds securely  
- **📢 IPO Center** — Discover and bet on new listings  
- **🌍 Global Market Access** — USA 🇺🇸 | India 🇮🇳 | Hong Kong 🇭🇰  
- **🧠 Quantitative Analysis** — Backtest and optimize trading strategies  

---

## 🛠 Tech Stack
- **Frontend:** React.js ⚛️, TypeScript 💻  
- **Styling:** Tailwind CSS 🎨  
- **Charts:** Chart.js 📊  
- **Data:** Yahoo Finance API
